# Nieuwe Eisen (Wachtrij)

## UI / Tray gedrag
- Voeg een knop toe in de indicator UI om het venster te verbergen (hide).
- Als de gebruiker in de taskbar/tray/menu-bar op VoicePaste klikt, moet de indicator weer zichtbaar worden.
- Voeg een herstart-knop toe in de indicator UI (klein vierkant knopje met een cirkelpijl) die de app afsluit en direct opnieuw start.
- Indicator openen linksboven in beeld (niet in het midden).
- Indicator is zichtbaar zodra de app start, niet alleen tijdens recording.

## Hotkeys
- Voeg een hotkey toe om recording te stoppen zonder transcription (opname annuleren).

## Notities
- Deze eisen zijn opgeslagen voor de volgende BRD (nog niet opgesteld).
